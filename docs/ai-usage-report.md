# AI Usage Report

## 1. Tools Used & Use Cases

### ChatGPT
I used ChatGPT to:
- plan how to upgrade Assignment 2 into Assignment 3
- generate and refine JavaScript logic for filtering, sorting, state management, and validation
- improve UI/UX structure and wording for user feedback messages
- help rewrite the README and technical documentation in a clearer and more professional format

### GitHub Copilot
I used GitHub Copilot to:
- speed up code writing through auto-completion
- suggest repeated HTML, CSS, and JavaScript patterns
- support small refactoring tasks while editing files

---

## 2. Benefits & Challenges

### Benefits
- AI helped speed up development and reduce repetitive coding work.
- It made it easier to think of stronger Assignment 3 features beyond the basic requirements.
- It supported debugging and helped organize the code into cleaner reusable functions.
- It improved documentation quality and made the final submission more professional.

### Challenges
- Some AI suggestions needed simplification because they were more complex than necessary.
- AI-generated code still required testing and review to make sure it matched the assignment requirements.
- A few suggestions needed manual correction to maintain code consistency and readability.

---

## 3. Learning Outcomes

By using AI tools in this assignment, I learned:
- how to connect a frontend application to an external API using `fetch`
- how to handle API errors in a user-friendly way
- how to build more advanced project filtering and sorting logic
- how to manage application state using `localStorage`
- how to improve code structure by splitting logic into small reusable functions
- how to review AI output critically instead of accepting it directly

---

## 4. Responsible Use & Modifications

To ensure correct and responsible AI use:
- I reviewed every AI-generated suggestion before adding it to the project.
- I modified the generated code to fit my own portfolio and assignment goals.
- I removed unnecessary complexity and adjusted naming, structure, and content.
- I checked that the final code remained understandable, original in presentation, and aligned with academic integrity.

AI was used as a **support tool for learning, productivity, and code review**, not as a replacement for my own understanding or effort.
